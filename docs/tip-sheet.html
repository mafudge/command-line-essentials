Path Tips & Naming Rules
Refer to my home folder ~  or  $env:USERPROFILE ~ ~ e.g. cd ~ or ls ~/Documents
Refer to current folder . . . a single dot
Refer to parent folder .. .. .. two dots
Use a path with spaces in the name cd "My Documents" cd "My Documents" cd "My Documents" wrap in quotes or escape with \
Path separator character \  (backslash) /  (forward slash) /  (forward slash) Windows also accepts / in many cases
Avoid these characters in names / \ : * ? " < > |  — also avoid spaces when possible use - or _ instead
