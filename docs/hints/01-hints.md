# Lesson 1 Hints

Stuck? These hints will guide you without giving away the answers.

---

## Exercise 1: Running Your First Command

<details>
<summary>Hint 1</summary>

The `echo` command prints text to the screen. You've seen it in the lesson!
</details>

<details>
<summary>Hint 2</summary>

Type exactly what you see in the lesson: `echo "Hello, World!"` including the quotes and exclamation mark.
</details>

<details>
<summary>Hint 3</summary>

After typing the command, press the Enter key on your keyboard to run it.
</details>

---

## Exercise 2: Finding Your Username

<details>
<summary>Hint 1</summary>

On macOS and Ubuntu, there's a command called `whoami` that tells you exactly who you are logged in as.
</details>

<details>
<summary>Hint 2</summary>

On Windows PowerShell, user information is stored in environment variables. The variable for username is `$env:USERNAME`.
</details>

<details>
<summary>Hint 3</summary>

The command should return just one word — your username. If you see something unexpected, try running it again to make sure you typed it correctly.
</details>

---

## Exercise 3: Clearing the Screen

<details>
<summary>Hint 1</summary>

Different operating systems use different commands for the same task. Windows uses a different command than macOS and Ubuntu.
</details>

<details>
<summary>Hint 2</summary>

For Windows, the command is short: `cls` (short for "clear screen").
</details>

<details>
<summary>Hint 3</summary>

For macOS and Ubuntu, the command is `clear`. If you've already typed some commands, try this one now — it should make your terminal look brand new.
</details>
