# Lesson 1 Exercises

Time to practice! These exercises will help you get comfortable with the terminal.

## Exercise 1: Hello, World!

**Goal:** Open your terminal and run your first command.

### The Command

<details>
<summary>Windows (PowerShell or Git Bash)</summary>

```powershell
echo "Hello, World!"
```
</details>
<details>
<summary>macOS (Terminal)</summary>

```bash
echo "Hello, World!"
```
</details>
<details>
<summary>Ubuntu (Terminal)</summary>

```bash
echo "Hello, World!"
```
</details>

### What to Look For

You should see the text `Hello, World!` printed on the screen immediately after you press Enter.

### Verify

<details>
<summary>Windows (PowerShell or Git Bash)</summary>

```powershell

```
</details>
<details>

</details>

## Exercise 2: Who Am I?

**Goal:** Find out your computer's username.

### The Commands

<details>
<summary>Windows (PowerShell or Git Bash)</summary>

```powershell
echo $env:USERNAME
```
</details>
<details>
<summary>macOS (Terminal)</summary>

```bash
whoami
```
</details>
<details>
<summary>Ubuntu (Terminal)</summary>

```bash
whoami
```
</details>

### What to Look For

You should see a single word — that's your username on this computer. It might be something like `john`, `admin`, or `maria`.

### Verify

<details>
<summary>Windows (PowerShell or Git Bash)</summary>

```powershell
echo $env:USERNAME
```
</details>
<details>
<summary>macOS (Terminal)</summary>

```bash
whoami
```
</details>
<details>
<summary>Ubuntu (Terminal)</summary>

```bash
whoami
```
</details>

## Exercise 3: Clear the Screen

**Goal:** Clear all text from your terminal screen.

### The Commands

<details>
<summary>Windows (PowerShell or Git Bash)</summary>

```powershell
cls
```
</details>
<details>
<summary>macOS (Terminal)</summary>

```bash
clear
```
</details>
<details>
<summary>Ubuntu (Terminal)</summary>

```bash
clear
```
</details>

### What to Look For

After running the command, your terminal should be empty — no old commands, no previous output. A fresh start.

<details>
